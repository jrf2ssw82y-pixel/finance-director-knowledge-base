# 合并报表编制参考手册

## 一、编制流程总览

```
收集单体报表
    ↓
调整子公司报表（非同控：公允价值；政策一致性调整）
    ↓
成本法→权益法调整（长期股权投资）
    ↓
填制合并工作底稿（母+各子公司报表加总）
    ↓
编制抵消分录：
  ① 投资与所有者权益抵消
  ② 投资收益与利润分配抵消
  ③ 内部债权债务抵消
  ④ 内部存货交易未实现利润抵消
  ⑤ 内部固定资产交易抵消
  ⑥ 现金流量表内部抵消
    ↓
生成合并资产负债表、利润表、所有者权益变动表、现金流量表
    ↓
平衡检验（借贷相等）
```

## 二、核心抵消分录模板

### 1. 投资与所有者权益抵消

```
借：子公司实收资本（子）
    资本公积——年初（子）
             ——本年（子）
    盈余公积——年初（子）
             ——本年（子）
    未分配利润——年末（子）
    商誉（=调整后长投 - 子公司权益×母%）
贷：长期股权投资（母，权益法调整后）
    少数股东权益（子公司权益×少%）
```

### 2. 投资收益与利润分配抵消

```
借：投资收益（= 子公司净利润 × 母%）
    少数股东损益（= 子公司净利润 × 少%）
    未分配利润——年初（子）
贷：提取盈余公积（子）
    对股东的分配（子，当年宣告股利）
    未分配利润——年末（子）
```

### 3. 内部应收/应付账款抵消

```
借：应付账款（内部）
贷：应收账款（内部）

借：应收账款——坏账准备（内部计提的坏账）
贷：资产减值损失
```

### 4. 内部存货交易未实现利润抵消

```
【当期购进】
借：营业收入（内部销售额）
贷：营业成本（内部销售额）

借：营业成本（未实现利润额）
贷：存货（未实现利润额）

【连续编制——上期递延】
借：未分配利润——年初
贷：营业成本
```

### 5. 内部固定资产交易抵消

```
【当期交易】
借：营业收入（含利润的售价）
贷：营业成本（账面价值部分）
    固定资产——原价（未实现利润部分）

借：固定资产——累计折旧（多计折旧的抵消）
贷：管理费用
```

### 6. 应付债券与持有至到期投资抵消

```
借：应付债券（期末摊余成本）
贷：持有至到期投资（期末摊余成本）

借：投资收益（期初摊余成本×实际利率）
贷：财务费用
```

## 三、持股结构分类处理

### 直接持股（P持有S 80%）

- P对S的长投：权益法调整后 → 与S的所有者权益抵消
- 少数股东权益 = S所有者权益 × 20%

### 间接持股（P→S 80%，S→T 70%）

**方法一：逐级合并**
1. 先合并 S+T，生成"S合并报表"
2. 再用"S合并报表"与 P 合并

**方法二：一次合并**
1. 直接用 P、S、T 三张报表同时合并
2. P对S的抵消用 P 对 S 的权益
3. S对T的抵消用 S 对 T 的权益（T的少数股东30%直接确认）
4. **注意**：P在T中的间接少数股东 = P持S 80% × S持T 70% = 56%

**P最终对T的权益**：
- 直接+间接 = 0 + 80% × 70% = 56%
- T的少数股东 = 30%（直接）+ 14%（间接通过S）= 合并少数股东44%

### 交叉持股（P持S 90%，S持P 10%）

**子公司持有母公司股份的处理**：
- S持有P的股份 → 在合并报表中作为"库存股"（所有者权益的减项）
- 调整公式：
  - 设 P合并净利润 = x
  - P自身净利润 + S净利润×90% = x
  - P净利润 = P自身 + 90% × (S自身 + 10% × x)
  - 解方程得 x

### 子公司相互交叉持股（B持C，C持B）

设联立方程：
```
B净利润 = B自身净利润 + B对C的投资收益
C净利润 = C自身净利润 + C对B的投资收益

B对C收益 = C净利润 × B持C比例
C对B收益 = B净利润 × C持B比例

联立求解
```

## 四、平衡检验要点

| 检验项 | 公式 |
|--------|------|
| 资产=负债+权益 | 资产总计 = 负债合计 + 少数股东权益 + 归母权益 |
| 净利润钩稽 | 合并净利润 = 归母净利润 + 少数股东损益 |
| 净利润 = 各实体自身净利润之和 | 抵消掉所有投资收益后的结果 |
| 抵消分录借贷相等 | 每张抵消分录借方合计 = 贷方合计 |
| 留存收益变动 | 期末未分配利润 = 期初 + 净利润 - 分配 |

## 五、常见错误排查

| 错误现象 | 可能原因 |
|----------|----------|
| 合并资产 ≠ 合并负债+权益 | 长投未完全抵消 / 少数股东权益计算有误 |
| 合并净利润偏大 | 投资收益未抵消 / 内部销售未抵消 |
| 未分配利润不平 | 年初数未递延 / 股利分配重复计算 |
| 少数股东权益异常 | 持股比例输错 / 连续编制未追溯调整 |

## 六、适用准则

- 《企业会计准则第33号——合并财务报表》（2014年修订）
- 《企业会计准则第2号——长期股权投资》
- 《企业会计准则解释第5号》（同一控制下合并）
