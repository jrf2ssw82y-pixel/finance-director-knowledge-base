## 财务总监职责

### **五大核心职责**
✅ **预算健康管理者**：确保公司财务状况稳健，评估ROI与风险  
✅ **风险控制者**：保持风险可控，避免高风险决策  
✅ **战略落地保障者**：确保战略可落地  
✅ **财务权限设计者**：设计谁可以看什么财务数据  
✅ **持续学习者**：不断进化，每天复盘学习，每周总结改进  

### **财务总监思维模式**
✅ **双向奔赴思维**：从财务看业务，又从业务看财务  
✅ **业财融合**：库存不仅仅是货，更是冻住的现金  
✅ **问题刨根问底**：为什么资金周转周期长了？为什么账上没钱？  
✅ **数据驱动决策**：最佳现金持有量模型量化资金需求  
✅ **预测导向**：现金流预算模型提前预测资金缺口  

## 财务总监六步框架与五大模型对应

✅ **看战略** → 现金流预算模型（预测未来资金需求）  
✅ **看成长** → 现金周转周期模型（资金周转效率）  
✅ **看效益** → 最佳现金持有量模型（资金使用效益）  
✅ **看效率** → 存货占用模型（库存管理效率）  
✅ **看资产质量** → 信用决策模型（应收账款质量）  
✅ **看风险** → 现金周转周期模型（资金周转风险）  
✅ **工作计划与建议** → 综合五大资金模型提出改进措施  

## CFO核心原则

✅ **Cash is oxygen** - 现金流是氧气  
✅ **13-week rolling forecast** - 13周滚动预测  
✅ **Raise when you can** - 能融资时就融资  
✅ **No board surprises** - 董事会无意外  
✅ **Every dollar has opportunity cost** - 每一美元都有机会成本  
✅ **Simplicity over precision** - 简单优于精确  
✅ **Finance enables** - 财务赋能  

## 实施建议五步法

✅ **第一步**：计算现金周转周期，与同行对比  
✅ **第二步**：设定最佳现金持有量上下限
